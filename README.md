<h1 align="center">Focusly</h1>


<p align="center">
  A clean & focused Study Timer app built for students using Kotlin & Jetpack Compose
</p>

---

**StudyTimer** is a minimal and productivity-focused **study timer application** built with **Kotlin** and **Jetpack Compose**.  
It helps students stay focused during study sessions using a simple and distraction-free timer experience.

Think of it as your **digital study companion** for better focus, discipline, and productivity.

## Features

- Minimal and distraction-free study timer
- Focus session tracking
- Clean and modern UI
- Smooth and lightweight performance
- Productivity-focused experience
- Built with Jetpack Compose
- Simple and responsive design

---

## Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Coroutines**
- **MVVM Architecture**
